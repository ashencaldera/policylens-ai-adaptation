# PolicyLens: AI-Driven Policy Adaptation Framework

PolicyLens is an intelligent system built to bridge the gap between high-level national strategies and local stakeholder needs. Specifically designed to process the **National AI Strategy for Sri Lanka 2024**, this tool automates the extraction, summarization, and contextual adaptation of complex policy mandates.

## 🚀 Overview

National policy documents are often dense and inaccessible to the general public. PolicyLens utilizes Generative AI to perform "Scenario Thinking," allowing users to generate tailored policy drafts for specific sectors like Education, Business, and Legal Governance.

## ✨ Key Features

- **NLP Preprocessing:** Custom cleaning module to normalize raw PDF data and remove artifacts.
- **Strategic Summarization:** Distills 60+ pages into core strategic pillars using Gemini 1.5 Flash.
- **Scenario-Based Adaptation:** Generates context-aware drafts for different stakeholders (SMEs, Educators, Lawyers).
- **Style Control:** Adjustable professional personas (Simple, Professional, Technical) via a slider interface.
- **Multilingual Support:** Instant translation into Sinhala and Tamil for local inclusivity.

## 🛠️ Tech Stack

- **Frontend:** [Streamlit]
- **AI Engine:** [Google Gemini API]
- **Language:** Python 3.9+
- **Data Handling:** PyPDF2, Regular Expressions (Regex)
